

# Open MV 4修改版

- 说明：参考自官方设计，主要修改的内容如下

	- ① PCB层数由4层板改为2层
	- ② TF卡改为了自弹式卡坐
	- ③ 摄像头连接座子改成了和OpenMV3一样的OV7725摄像头模组（感光芯片+镜头）

- 官方开源设计链接：https://github.com/openmv

- EDA设计软件：Altium Designer 20
